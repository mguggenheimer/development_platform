# development_platform
scripts and configs to set up containerized/kvm development spaces on a linux based virtual machine

# kvm_setup.sh
chmod +x kvm_setup.sh && ./kvm_setup.sh

# dev_sandbox
~/dev-sandbox$ code .

Once code loads, click "reopen in dev container"
